# Certificates Folder

This folder is where you can store your certificate images to display on your portfolio.

## How to add certificates:

1. Add your certificate images (PNG, JPG, or PDF screenshots) to this folder
2. Update the portfolio.html file to display your certificates
3. Recommended image formats: PNG or JPG
4. Recommended image size: 800x600 pixels or similar aspect ratio

## Example certificate display code:

Replace the certificate placeholder in portfolio.html with:

```html
<div class="certificate-item">
    <img src="certificates/your-certificate.png" alt="Certificate Name">
    <h3>Certificate Title</h3>
    <p>Issuing Organization</p>
    <p>Date Earned</p>
</div>
```

## Supported file types:
- .png
- .jpg
- .jpeg
- .pdf (as screenshots)
