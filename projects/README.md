# Projects Folder

This folder is where you can store your project screenshots to display on your portfolio.

## How to add projects:

1. Add your project screenshots (PNG, JPG) to this folder
2. Use descriptive filenames (e.g., "ecommerce-website.png", "inventory-system.jpg")
3. Recommended image size: 1200x800 pixels or 16:9 aspect ratio
4. Include both desktop and mobile views if applicable

## Project naming convention:
- Use lowercase letters and hyphens
- Be descriptive: `project-name-screenshot.png`
- Examples:
  - `ecommerce-website.png`
  - `inventory-management-system.jpg`
  - `restaurant-pos-system.png`
  - `blog-cms-dashboard.jpg`

## Supported file types:
- .png (recommended for screenshots)
- .jpg
- .jpeg

## Tips:
- Take clean, professional screenshots
- Show the most important features
- Include both frontend and backend views if applicable
- Consider adding mobile responsive views
